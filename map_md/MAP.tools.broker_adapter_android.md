# MAP.tools.broker_adapter_android

## broker-adapters/android
- broker-adapters/android/package.json - Declares the Android adapter package metadata.
- broker-adapters/android/tsconfig.json - Configures TypeScript compilation for the Android adapter.
- broker-adapters/android/src/index.ts - Re-exports the React adapter for the JS side and defines Android-specific diagnostics (Metro status, Logcat reference, device info) with registration helpers.
