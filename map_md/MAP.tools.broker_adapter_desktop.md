# MAP.tools.broker_adapter_desktop

## broker-adapters/desktop
- broker-adapters/desktop/package.json - Declares the desktop adapter package metadata.
- broker-adapters/desktop/tsconfig.json - Configures TypeScript compilation for the desktop adapter.
- broker-adapters/desktop/src/index.ts - Re-exports the React adapter for the renderer process and defines main-process diagnostics (logs, IPC status, preload status) with registration helpers.
