# MAP.root.scraping_api

## root
- package.json - Declares the Node scraping proxy package metadata, scripts, and runtime dependencies.
- package-lock.json - Locks the installed dependency graph for the Node scraping proxy.
- server.js - Implements the NetShort and Dramafren scraping API routes and parsing helpers.
- README.md - Documents the scraping API endpoints and local usage examples.
- base-agent.md - Provides the Vietnamese base workflow for building project-specific AI agents.
- Map.role.vietnamese.md - Defines the Vietnamese guidance for the repository map system.
- React.role.vietnamese.md - Defines the Vietnamese React role for component structure, logs, and investigation workflow.
- detail_sample.html - Stores a NetShort detail-page capture for episode parsing research.
- watch_ep1.html - Stores a NetShort watch-page capture for stream URL extraction research.
- dramafren_home.html - Stores a Dramafren home-page capture for series-list parsing research.
- dramafren_series_sample.html - Stores a Dramafren series-page capture for episode-list parsing research.
- dramafren_watch_sample.html - Stores a Dramafren watch-page capture for mirror parsing research.
- .github/agents/base.agent.md - Holds the current placeholder custom agent definition for the workspace.
- .github/roles/Map.role.md - Defines the English role for maintaining the workspace map index and child maps.
